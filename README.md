# Versal Pulse - Team Productivity Dashboard

A modern, real-time team productivity monitoring and management dashboard built with React, Redux, and Tailwind CSS.

## 🌐 Live Demo

**[🚀 View Live Application](https://versal-pulse.vercel.app/)**

## 📹 Video Walkthrough

**[🎥 Watch Demo Video](https://drive.google.com/file/d/1Wo-VIPxeStKbItYOPworXESHxH1dM4ic/view)** - Get a comprehensive overview of all features and functionality

## 🚀 Key Features

- **Real-time Team Monitoring**: Track team member status and productivity
- **Role-based Access**: Separate interfaces for Team Leads and Team Members
- **Task Management**: Assign, track, and manage tasks with progress monitoring
- **Notification System**: Toast notifications and notification center with sound alerts
- **Analytics Dashboard**: Team performance analytics and charts
- **Calendar Integration**: Team scheduling and deadline management
- **Responsive Design**: Works on desktop and mobile devices

## 👥 User Roles

### Team Lead
- Team dashboard with member status overview
- Task assignment and management
- Analytics and performance metrics
- Team member details and performance tracking
- Calendar and scheduling features

### Team Member
- Personal dashboard with task overview
- Status updates (Working, Break, Meeting, Offline)
- Task progress tracking and updates
- Personal productivity monitoring

## 🛠️ Tech Stack

- **React 19.1.1** with hooks and functional components
- **Redux Toolkit** for state management
- **Tailwind CSS** for styling
- **Chart.js** for data visualization
- **Vercel** for deployment

## 🚀 Quick Start

```bash
# Clone repository
git clone <repository-url>
cd Versal-Pulse

# Install dependencies
npm install

# Start development server
npm start
# or
npm run dev

# Open http://localhost:3000 or http://localhost:3001
```
